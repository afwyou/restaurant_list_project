## Installation
#### clone this project
```bash
   $ git clone https://github.com/afwyou/restaurant_list_project.git
```
#### Install npm Express
```bash
   $ npm i express
```
#### cd to project folder
```bash
   cd restaurant_list_project
   ```
#### Require seeder
```bash
   $ npm run seed
   ```
#### Run this app
```bash
   $ npm run dev
   ```
If terminal shows below content means it's working
```bash
   Listen to the http://localhost:3000
   ```
